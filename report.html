  • what my app does

    searches the spotify api for a musician that is inputted by the user via form. it then gets the top tracks of the artist and sends it to a wordcloud api that creates a word cloud using the titles of the top tracks.

  • deployed

    deployed via github pages.

  • is responsive

    smaller window will make the search bar smaller, and only display one wordcloud per line.

  • uses 2 different apis

    this uses the spotify api and a word cloud api.

  • has search

    ability to search for an artist.

  • displays results

    displays the word cloud.

  • results are filterable

    you can remove word clouds you no longer want to see with a button.
